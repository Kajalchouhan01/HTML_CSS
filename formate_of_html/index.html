About HTML

Hypertext Markup Language, or HTML, is a programming language used to describe the structure of web pages. HTML makes it possible to create static pages with text, headings, tables, lists, images, links, and so on.


HTML has a lot of use cases, namely:

  • Web development
  • Internet navigation
  • Web documentation

The three main part of an html element are:

  1. Opening tag : used to state where an element starts to take effect. The tag is wrapped with opening and closing angle brackets. For example, use the start tag
  2. Content : this is the output that other users see.
  3. Closing tag : the same as the opening tag, but with a forward slash before the element name
